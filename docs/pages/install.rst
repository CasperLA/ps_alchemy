Installing
==========

GitHub
------

.. code-block:: bash

    pip install git+http://github.com/sacrud/ps_alchemy.git

PyPi
----

.. code-block:: bash

    pip install ps_alchemy

Source
------

.. code-block:: bash

    git clone git+http://github.com/sacrud/ps_alchemy.git
    cd ps_alchemy
    pip install .

Develop
-------

.. code-block:: bash

    git clone git+http://github.com/sacrud/ps_alchemy.git
    cd ps_alchemy
    pip install -e .
